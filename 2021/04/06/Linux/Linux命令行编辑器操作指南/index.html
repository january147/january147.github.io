Linux命令行编辑器操作指南

Posted on 2021-04-06 Edited on 2021-01-11 In Linux

Linux命令行编辑器操作指南

vim

在命令模式使用a或者i进入插入模式，a从当前字符后开始，i从当前字符前开始。

在插入模式使用ESC进入命令模式

命令模式

命令 描述
yy/yw 拷贝一行/拷贝一个词
p 粘贴
dd/dw 删除一行/删除一个词
gg 跳转到文件头
G 跳转到文件头
^ 跳转到行首
$ 跳转到行尾
w 行内后移个单词(word)
b 行内前移（backward）个单词

字符搜索和替换

命令 描述
/<关键字> 查找关键字
:%s/<关键字1>/<关键字2>/gc 将关键字1替换为关键字2
:2,23s/<关键字1>/<关键字2>/gc 指定替换范围为2,23

窗口操作

命令 描述
ctrl+w(进入窗口命令) 按下ctrl+w再按下述字符执行对应命令
w 在窗口间按顺序跳转
h/j/k/l 在窗口间按方向（左/下/右/上）跳转
| 放大水平方向主要窗口（左右分屏后的主要窗口）
_(下划线) 放大垂直方向的主要窗口（上下分屏后的主要窗口）
= 恢复原窗口大小
命令 描述
:split 上下分屏
:vslipt 左右分屏

插入模式

emac

gnu nano

^表示ctrl

M表示alt

sed

选项 描述
-i 直接在文件中替换（未指定该选项则将替换后的文件输出到标准输出）

sed脚本命令

flags 功能
n 1~512 之间的数字，表示指定要替换的字符串出现第几次时才进行替换，例如，一行中有 3 个 A，但用户只想替换第二个 A，这是就用到这个标记；
g 对数据中所有匹配到的内容进行替换，如果没有 g，则只会在第一次匹配成功时做替换操作。例如，一行数据中有 3 个 A，则只会替换第一个 A；
p 会打印与替换命令中指定的模式匹配的行。此标记通常与 -n 选项一起使用。
w 将缓冲区中的内容写到指定的 file 文件中；
& 用正则表达式匹配的内容进行替换；
\n 匹配第 n 个子串，该子串之前在 pattern 中用 () 指定。
\ 转义（转义替换部分包含：&、\ 等）。
