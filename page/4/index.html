椭圆曲线数学

Posted on 2021-04-06 Edited on 2020-12-21 In 密码学数学基础

椭圆曲线数学基础

椭圆曲线种类

short Weierstrass格式（最常用）

$$
y^2 = x^3 + ax + b
$$

Montogomery格式

$$
by^2 = x^3 + ax^2 + x
$$

Edward格式

$$
x^2 + y^2 = 1 + dx^2y^2
$$

椭圆曲线点的坐标格式

仿射坐标（affine coordinates）

仿射坐标下的椭圆曲线点坐标为$$(x,y)$$

标准射影坐标

标准射影坐标下椭圆曲线的点坐标为$$(x,y,z)$$

Jacobian射影坐标

双线性对

常用术语和符号

符号 描述
$$F_{q^k}$$ $$F_{q}$$的$$k$$次扩域
$$E(F_{q^k})$$ $$F_{q^k}$$上的椭圆曲线（群）

基域和其对应扩域

基域通常用$$F_q$$表示，其对应$k$次扩域用$$F_{q^k}$$表示，如果我们将基域中元素视为标量，则扩域中元素相当于n维向量，向量每个维度的值可以使用基域中元素来表示。

扭曲线

对于椭圆曲线$$E(F_{q^k})$$，其对应的$d$次扭曲线为定义在$F_{q^{k/d}}$上的椭圆曲线$E’(F_{q^{k/d}})$，曲线$E$上的点可以用$E’$上的点表示。

Miller算法

简要描述

首先确定运算域$$F_{q^k}$$，以及其上的椭圆曲线$$E(F_{q^k})$$，其中$$k$$是椭圆曲线的相对于参数$$r$$的嵌入次数。在椭圆曲线上的取两点（$$U,V$$）构成一条$$F_{q^k}$$上的直线$$ax+by+c=0$$，以该直线构造一个函数
$$
g_{U,V}(Q) = ax_Q + by_Q + c
$$
该函数接受椭圆曲线的一个点作为输入，输出$$F_{q^k}$$上的一个值。通过该函数可以将椭圆曲线群$$E(F_{q^k})$$中的点映射到$$F(q^k )$$中的值。Miller算法用于计算Miller函数，Miller函数如下：
$$
y=f_{P,c}(Q)
$$
其应该有3个输入参数，其中$P,Q$是椭圆曲线$E(F_{q^k})$上两点，$c$是一个普通整数，输入值$$y$$是$$F_{q^k}$$中的元素。Miller函数将被广泛用于双线性对的构造。

密码学标准

Posted on 2021-04-06 Edited on 2020-12-17 In 开发者密码学

密码学标准

Public Key Cryptography System(PKCS)

PKCS#1主要定义了RSA加密签名的算法、密钥格式、以及加密签名数据格式。

Version RFC 编号 简介
PKCS #1 RFC 8017 RSA Cryptography Standard（RSA加密签名模式以及公私钥格式）包括加密方案RSAES-OAEP, RSAES-PKCS1-V1.5, 签名方案RSASSA-PSS, RSASSA-PKCS1-V1.5
PKCS #5 RFC 8018 Password-based Encryption Standard
PKCS #7 RFC 2315 Cryptographic Message Syntax Standard（包括了加密/签名数据格式，分组密码填充格式）
PKCS #8 RFC 5958 RFC 5208 Private-Key Information Syntax Standard (包括了通用的私钥文件格式). RFC5958对RFC5208定义的私钥格式进行了一些修改，主要是支持了包含公钥内容，因此RFC5958的私钥格式和RFC5208的私钥格式版本号分别为1和0.
PKCS #10 RFC 2986 Certification Request Standard
PKCS #12 RFC 7292 Personal Information Exchange Syntax Standard

椭圆曲线公钥密码

Standards for Efficient Cryptography（SEC）定义了椭圆曲线公钥密码的各种标准，其中SEC 1中定义了椭圆曲线私钥格式，同RFC 5159定义的椭圆曲线私钥格式一致，通常称其为“sec1格式私钥”。

标准化组织

ITU（国际电信联盟）

国际电信联盟是联合国的一个重要专门机构，也是联合国机构中历史最长的一个国际组织。简称“国际电联”、“电联”或”ITU”。

国际电联是主管信息通信技术事务的联合国机构，负责分配和管理全球无线电频谱与卫星轨道资源，制定全球电信标准，向发展中国家提供电信援助，促进全球电信发展。

作为世界范围内联系各国政府和私营部门的纽带，国际电联通过其麾下的无线电通信、标准化和发展电信展览活动，而且是信息社会世界高峰会议的主办机构。

国际电联总部设于瑞士日内瓦，其成员包括193个成员国和700多个部门成员及部门准成员和学术成员。每年的5月17日是世界电信日(World Telecommunication Day)

—-引用自百度百科

IETF（互联网工程任务组）

互联网工程任务组，成立于1985年底，是全球互联网最具权威的技术标准化组织，主要任务是负责互联网相关技术规范的研发和制定，当前绝大多数国际互联网技术标准出自IETF
